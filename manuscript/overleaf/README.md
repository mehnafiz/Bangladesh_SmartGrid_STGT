# manuscript/overleaf/

**Purpose**
Self-contained, Overleaf-compatible LaTeX project for the paper.

**Future Contents**
- `sections/` — modular `.tex` section files.
- `figures/` — figures included in the paper.
- `tables/` — LaTeX tables.
- `bibliography/` — `.bib` reference files.
- `styles/` — class / style files.
- `journal_template/` — target journal template.
