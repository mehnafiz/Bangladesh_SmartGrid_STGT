# src/

**Purpose**
Core, importable source code for the project. All reusable logic for data
handling, modeling, training, evaluation, and explainability lives here.

**Future Contents**
- `data/`, `preprocessing/`, `features/`, `datasets/` — data pipeline.
- `graph/` — graph construction utilities.
- `models/`, `losses/`, `metrics/` — model components.
- `training/`, `evaluation/` — training and evaluation loops.
- `explainability/`, `visualization/` — interpretation and plotting.
- `utils/` — shared helpers.
