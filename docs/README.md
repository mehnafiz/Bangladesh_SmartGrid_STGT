# docs/

**Purpose**
Living documentation for the research project: phase planning, design
reviews, architecture, methodology, and traceable decision records.

**Future Contents**
- `phases/` — per-phase plans and progress notes.
- `reviews/` — internal and peer review records.
- `architecture/` — system and model architecture docs.
- `methodology/` — research methodology documentation.
- `meeting_notes/` — meeting summaries and actions.
- `decision_logs/` — recorded design decisions.
- `checklists/` — reproducibility and quality checklists.
